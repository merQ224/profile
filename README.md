# my-profile

Frontend Single Page Application (SPA) built with **React** and **Vite** to explore modern **React** architecture, component design, and frontend tooling.

This project serves as a personal portfolio site showcasing my background, career experience, and projects while demonstrating clean structure and modern React development practices.

---

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Git

---

## Features

- Component based architecture
- Reusable UI components
- Clean folder structure
- Responsive layout

---

## Project Structure

src/
- components/ → Reusable UI components
- pages/ → High-level sections (Landing, About, Career, Projects)
- data/ → Static content separated from UI
- hooks/ → Custom reusable logic
- assets/ → Images and static files
- main.jsx → Application entry point
- App.jsx → Root component composition

## Getting Started

### 1. Clone Repository

git clone https://github.com/merQ224/profile.git
cd my-profile

### 2. Install Dependencies

npm install

### Run

npm run dev
open http://localhost:5173

---

## Production Build

### Optimised production build

npm run build
- Build output will be generated in dist/ folder

### Preview production build (locally)

npm run preview
